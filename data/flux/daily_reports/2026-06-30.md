## 4. Site-level SHAP plots with 3 disturbance metrics

Created `plot_21_site_shap_distmetrics.R` and new output folder `plots/site_shap_disturbance_metrics/`.

**Scope**: M04 (C+T+D), 12m and 24m windows, all 5 EFPs (GPPsat, NEPmax, ETmax, uWUE, WUE)

**Layout**: 3-panel patchwork per plot — same stacked SHAP bars in each panel, dots coloured by a different disturbance metric

**Driver groups shown**: Climate, Traits, Disturbance (Memory removed; re-normalized to 3 groups)

**Disturbance metrics (all peak value across years per site, 500m radius)**:

| Panel | Metric | Formula | Range |
|-------|--------|---------|-------|
| 1 | Absolute Mortality | `deadwood_mean_pct_500m` | 0–42% |
| 2 | Relative Mortality | `(deadwood / forest) × 100` | 0–81% |
| 3 | Relative Disturbance | `(deadwood + loss×100) / (forest + loss×100) × 100` | 0–100% |

**Color scale**: green (low) → orange → red (high), midpoint at 10% (abs/rel mortality) and 20% (rel disturbance). Red threshold = 20% for all metrics.

**Relative disturbance formula rationale**: denominator uses original forest extent (current forest + already lost forest), so the metric is bounded 0–100% and reads as "fraction of original forest showing disturbance signal (standing deadwood + forest loss)". Previous version divided by current forest only, allowing values >100%.

**Outputs** (20 files = 10 PNG + 10 PDF):
- `site_shap_M04_{EFP}_{12m|24m}_distmetrics.{png|pdf}`
